=========
rv_output
=========

:envvar:`rv_output` represents the top level of the directory tree of
RVSpecfit outputs.

Different RVSpecFit outputs are located in different RVRUN folders corresponding to different runs of the pipeline. In this release the RVRUN=240520 contains
measurements from stacked desi spectra, while RVRUN=240521 contains single epoch measurements.
